# BinaryDiffViewer

BinaryDiffViewer は、Windows 向けの WPF バイナリ比較ツールです。  
2 つのバイナリファイルを 16 進数形式で表示し、差分の確認・一覧表示・CSV 出力を行えます。

## 目的と現在の実装状況

| 項目 | 状況 | 補足 |
| --- | --- | --- |
| バイナリファイルを開いて内容を確認する | 実装済み | ファイル A / B を個別に読込可能 |
| 2 つのバイナリファイルを比較する | 実装済み | 非同期比較、進捗表示、キャンセル対応 |
| ファイル内容を 16 進数形式で表示する | 実装済み | `0x00000000` 形式の Offset、Hex、ASCII を表示 |
| 差分バイトを分かりやすく表示する | 実装済み | 差分バイトを `[]` で強調表示 |
| 差分一覧を表示する | 実装済み | Offset、File A Byte、File B Byte、Diff Type を表示 |
| 将来的には大きなファイルも比較できるようにする | 一部対応 | 比較はストリーム処理、表示はフォーカス範囲表示 |
| 比較結果を CSV などで出力できるようにする | 一部対応 | CSV 出力あり。現状は画面表示中の差分一覧ベース |

## 主な機能

- ファイル A / ファイル B の読込
- Hex Viewer による 16 進数 + ASCII 表示
- 左右 Hex Viewer の縦横スクロール同期
- 差分一覧クリックによる該当 Offset 付近へのジャンプ
- 差分件数、先頭差分 Offset、ファイルサイズ一致/不一致の表示
- 選択中の差分 Offset の表示
- 差分一覧の CSV 出力
- 比較処理のキャンセル

## 動作環境

- Windows
- .NET 9 SDK
- WPF

## ビルド

```powershell
dotnet build .\BinaryDiffViewer.sln -nologo
```

Visual Studio から `BinaryDiffViewer.sln` を開いてビルド/実行しても構いません。

## 実行

ビルド後に次の実行ファイルを起動してください。

```text
bin\Debug\net9.0-windows\BinaryDiffViewer.exe
```

## 使い方

1. `ファイルAを開く` と `ファイルBを開く` で比較対象のファイルを選択します。
2. `比較` を押して差分比較を開始します。
3. 上部の状態バーで比較状態、差分総数、最初の差分、ファイルサイズ比較を確認します。
4. 左右の Hex Viewer で内容を比較します。スクロールは左右で同期します。
5. 下部の差分一覧で行を選択すると、その Offset を含む行へ左右の Hex Viewer が移動します。
6. 必要に応じて `結果をCSV出力` から差分一覧を保存します。

## 画面構成

- **上部バー**: 比較状態、進捗、差分総数、最初の差分、サイズ比較、選択中の差分
- **中央エリア**: ファイル A / B の Hex Viewer
- **下部エリア**: 差分一覧 DataGrid

## 既知の制約

- 差分一覧の表示件数には上限があります（現状 10,000 件）。
- CSV 出力は現在、画面表示に保持している差分一覧をもとに出力します。
- 比較処理自体はファイル全体を対象にストリーム比較しますが、Hex Viewer は全文仮想化ではなく、表示対象範囲を再生成する方式です。
- 初回にファイルを開いた直後の Hex 表示は先頭 1MB のプレビューです。差分一覧を選択すると、その差分 Offset 周辺の表示へ切り替わります。

## プロジェクト構成

- `Commands\RelayCommand.cs`  
  シンプルな `ICommand` 実装
- `Models\`  
  差分結果や差分種別を表すモデル
- `Services\`  
  ファイル読込、比較処理、Hex 整形、CSV 出力
- `ViewModels\`  
  画面状態と表示用データの管理
- `MainWindow.xaml` / `MainWindow.xaml.cs`  
  UI 定義と、スクロール同期・差分ジャンプなどの UI 依存処理
